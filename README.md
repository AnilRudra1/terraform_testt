# terraform_testt

This repository was cloned from https://github.com/AnilRudra1/terraform_testt.git and initialized locally with a minimal Terraform example.

Quickstart

1. Install Terraform: https://www.terraform.io/downloads
2. Initialize:

```bash
cd terraform_testt
terraform init
```

3. Validate and plan:

```bash
terraform validate
terraform plan
```

This repo currently contains a minimal `main.tf` placeholder.
